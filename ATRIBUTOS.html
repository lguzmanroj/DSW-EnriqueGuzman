ATRIBUTOS

Los elementos en HTML tienen atributos; estos son valores adicionales que configuran los elementos o ajustan su comportamiento de diversas formas para cumplir los criterios de los usuarios.

Lista de atributos

accept form, input Lista de tipos aceptados por el servidor, usualmente un tipo de archivo.
align (applet, caption, col, colgroup, hr, iframe, img, table, tbody, td, tfoot , th, thead, tr) Especifica el alineamiento horizontal del elemento.
border (img, object, table) El ancho del borde.
class (Atributo Global) Usualmente empleado con CSS para aplicar estilos a elementos con propiedades en comun.
color (basefont, font,hr) Este atributo establece el color de texto utilizando un nombre de color o en formato hexadecimal #RRGGBB.
height (canvas, embed, iframe, img, input, object, video/video) Especifica la altura de los elementos enumerados aquí. Para todos los demás elementos, usa la propiedad CSS height en su lugar.
sandbox (iframe /iframe) Lista de restricciones a ser desactivadas en el iframe.
size (input, select/select) Define el ancho del elemento (en píxeles). Si el atributo del elemento es del tipo text o password entonces es el número de caracteres.
src (audio, embed, iframe, img, input, script, source, track, video) La URL del contenido integrable.
width (canvas, embed, iframe, img, input, object, video) Para los elementos enumerados aquí, esto establece el ancho del elemento.
